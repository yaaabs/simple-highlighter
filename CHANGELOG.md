# Changelog

All notable changes to this project are documented here using semantic versioning.

## [1.1.1] - 2026-05-17

### Added

- Modernized popup UI with a more premium layout.
- Dedicated `src/` folder structure for JavaScript, HTML, and CSS.
- MIT License for open-source distribution.
- Improved README for users, developers, and release guidance.

### Changed

- Updated the popup frame to feel more rectangular and balanced.
- Strengthened validation messages for empty and non-highlighted erase actions.
- Improved extension description copy for store listings and discovery.

### Removed

- Shortcut feature UI and command wiring for a simpler extension experience.
